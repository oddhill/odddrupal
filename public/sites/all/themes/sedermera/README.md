# Odd baby

A starter theme for Drupal theming.  

### Dependencies
* Grunt (included)
* Sass 3.3
* Grunt CLI
* Bourbon(included)
* Neat(included)
* modernizr(included) - Support for csstransform 2D & 3D, svg touch & includes SHIV.
* selectivizr(included)
